# Check third party licences

Install `yolk3k` pip package:

    pip install yolk3k

Then execute command:

    yolk -l -f license

Output will look like:

```
Babel (2.2.0)
    License: BSD

Beaker (1.6.4)
    License: BSD

CherryPy (3.6.0)
    License: BSD

FormEncode (1.3.0a1)
    License: PSF

Genshi (0.7)
    License: BSD
...
```
